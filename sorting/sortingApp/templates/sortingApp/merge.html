Time and Space Complexity Analysis

  • Time Complexity: O(n log(n))

    • n is the length of the input array.

    • Log n is how many recursive calls we make. Since we split the array in half, the number of recursive calls is O(log(n)).

  • Space Complexity: O(1)

    • The larger the size of our input array, the greater the number of subarrays we must create in memory.Therefore, Merge Sort has a linear space complexity, O(n).

  • When should we use Bubble Sort?

    • We have unlimited memory available.
