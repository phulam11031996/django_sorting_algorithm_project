Time and Space Complexity Analysis

  • Time Complexity: O(n log(n))

    • Average Case: O(n log(n)). When we chose a random pivot.

    • Worst Case: O(n*n). When we chose the worst pivot.

    • Best Case: O(nlog(n)). When we chose the median pivot.

  • Space Complexity: O(1)

    • Our implementation of quickSort uses O(n) space because of the partition arrays we create. Note: There is also another version uses O(log(n)) space.

  • When should we use Bubble Sort?

    • The recursive code is light and simple to implement than merge sort.

    • When we know the constrain of data set. Since the algorithm allows us to pick the pivot anywhere in the array.
