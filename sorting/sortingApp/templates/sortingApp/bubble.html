Time and Space Complexity Analysis

  • Time Complexity: O(n*n)

    • O(n*n). Worst case scenario when input array is completely unsorted.

    • O(n). Best case scenario when input array is fully sorted. The nested loop will only have to perform one single pass.

  • Space Complexity: O(1)

    • Bubble Sort is a constant space, O(1), algorithm. The amount of memory consumed by the algorithm does not increase relative to the size of the input array.

  • When should we use Bubble Sort?

    • When sorting really small arrays where run time will be negligible no matter what algorithm we choose.

    • When sorting arrays that we expect to already be nearly sorted.
